Assignment Extension Manager (LTI 1.3)

This tool allows students to request deadline extensions for assignments and instructors to review and approve/deny those requests.

LTI 1.3 Configuration

  • Launch URL: {{ url_for('lti_launch', _external=True) }}
  • JWKS URL: {{ url_for('lti_jwks', _external=True) }}

Features

  • Student view: Request extensions with supporting documentation
  • Instructor view: Review requests, approve/deny with notes
  • Policy management: Set extension limits and requirements
  • Document upload and viewing capabilities
